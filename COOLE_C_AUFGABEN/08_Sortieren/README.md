# 08 Sortieren

## Ziel
Schreibe ein Programm, das ein Array mit Bubble Sort sortiert.

## Anforderungen
- Lege ein Array mit 5 Zahlen an.
- Gib das Array vor dem Sortieren aus.
- Sortiere das Array mit Bubble Sort.
- Gib das Array nach dem Sortieren aus.

## Dateien
- `main.c`

## Tipps und Tricks
- Vergleiche immer zwei benachbarte Werte.
- Wenn der linke Wert größer ist, werden die Werte getauscht.
- Für Bubble Sort brauchst du zwei Schleifen.

## Lösungsidee
- Lege das Array mit festen Werten an.
- Gib es zuerst mit einer Schleife aus.
- Sortiere es mit zwei `for`-Schleifen.
- Gib das Ergebnis erneut aus.
