# 07 – Chaoskonsole Deluxe

## Story
Eine kaputte Konsole soll drei Zahlen anzeigen.

## Ziel
Übe einfache Funktionen.

## Themen
- Funktionen
- Parameter
- Rückgabewerte

## Aufgabe

1. Schreibe eine Funktion, die zwei Zahlen addiert.
2. Schreibe eine Funktion, die die größere von zwei Zahlen findet.
3. Rufe beide Funktionen im `main` auf.
4. Gib die Ergebnisse aus.

## Bonus
- Schreibe noch eine Funktion für das Produkt von zwei Zahlen.

## Beispiel
```text
Summe: 9
Groessere Zahl: 5
