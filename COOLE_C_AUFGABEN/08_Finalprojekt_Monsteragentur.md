# 08 – Finalprojekt Monsteragentur

## Story
Du verwaltest eine kleine Liste von Monstern.

## Ziel
Verbinde mehrere einfache Themen in einer Aufgabe.

## Themen
- Structs
- Arrays
- Funktionen
- Schleifen

## Aufgabe

1. Definiere:
```c
struct Monster {
    char name[20];
    int scare_level;
};
```

2. Lege ein Array mit 3 Monstern an.
3. Gib alle Monster aus.
4. Finde das Monster mit dem höchsten `scare_level`.

## Bonus
- Sortiere die 3 Monster nach `scare_level`.

## Beispiel
```text
Momo - 4
Zorg - 9
Boo - 6
Staerkstes Monster: Zorg
