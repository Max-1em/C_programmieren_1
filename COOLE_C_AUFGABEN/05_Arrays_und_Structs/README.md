# 05 Arrays und Structs

## Ziel
Schreibe ein Programm, das mehrere Datensätze in einem Array speichert.

## Anforderungen
- Definiere ein `struct Student` mit `name` und `age`.
- Lege ein Array mit 3 Studenten an.
- Gib alle Studenten aus.
- Berechne das Durchschnittsalter.
- Gib den ältesten Studenten aus.

## Dateien
- `main.c`

## Tipps und Tricks
- Ein `struct` bündelt mehrere Werte.
- Mit einem Array kannst du mehrere `struct`-Elemente speichern.
- Eine `for`-Schleife eignet sich gut für die Ausgabe.

## Lösungsidee
- Definiere zuerst das `struct`.
- Lege dann ein Array mit festen Werten an.
- Laufe mit einer Schleife durch das Array.
- Vergleiche die Alter, um das Maximum zu finden.
