# 03 Schleifen und Zahlen

## Ziel
Schreibe ein Programm, das eine positive ganze Zahl einliest und auswertet.

## Anforderungen
- Lies eine positive Zahl ein.
- Berechne die Anzahl der Ziffern.
- Berechne die Quersumme.
- Gib beide Werte aus.

## Dateien
- `main.c`

## Tipps und Tricks
- Mit `% 10` erhältst du die letzte Ziffer.
- Mit `/ 10` entfernst du die letzte Ziffer.
- Eine `while`-Schleife ist hier gut geeignet.

## Lösungsidee
- Speichere die Eingabe in einer Variablen.
- Arbeite mit einer Kopie der Zahl.
- Wiederhole die Berechnung, bis die Zahl 0 ist.
